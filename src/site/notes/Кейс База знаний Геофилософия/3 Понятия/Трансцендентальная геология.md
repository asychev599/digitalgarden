# Трансцендентальная геология

**Трансцендентальная геология** — это незавершенный проект позднего [[Кейс База знаний Геофилософия/4 Авторы/Мерло-Понти, Морис\|Мориса Мерло-Понти]], изложенный в его рабочих записях к курсу о Природе.

## Суть проекта
- **Цель**: Исследовать [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Землю]] как **трансцендентальное условие** всякого опыта, особенно телесного. Это попытка создать [[Кейс База знаний Геофилософия/2 Темы/Геофеноменология\|Геофеноменология]].
- **[[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земля]] как гиперболизированная телесность**: [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земля]] для Мерло-Понти — это не просто объект, а **"[[Кейс База знаний Геофилософия/3 Понятия/Плоть мира\|Плоть мира]]"**, тотальное бытие тела, которое является условием возможности нашего собственного телесного опыта.

## Ключевые идеи
- **[[Кейс База знаний Геофилософия/3 Понятия/Феноменология как археология\|Феноменология как археология]]**: Мерло-Понти предлагает исследовать опыт как геолог исследует землю — вскрывая **"слои"** восприятия, чтобы добраться до самых изначальных, до-рефлексивных структур.
- **[[Кейс База знаний Геофилософия/3 Понятия/Оседание опыта\|Оседание опыта]]**: Глобальный, геологический масштаб опыта [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земли]] (ее [[Кейс База знаний Геофилософия/3 Понятия/Глубинное время (Deep Time)\|Глубинное время (Deep Time)]], ее "плотская про-история") **"оседает"** во множестве локальных, индивидуальных телесных перспектив.
- **Сплетение времени и пространства**: [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земля]] — это место, где происходит изначальное **сплетение** времени и пространства. "Схватить то самое время, которое есть пространство" — одна из задач трансцендентальной геологии.
- **Невидимое**: [[Кейс База знаний Геофилософия/3 Понятия/Трансцендентальная геология\|Трансцендентальная геология]] — это исследование **невидимого** основания всего видимого. [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земля]] — это "невидимая сила, позволяющая быть всему видимому".
