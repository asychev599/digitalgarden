# Политическая земля

**Политическая земля** — это концепция, рассматривающая [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Землю]] не как пассивный ресурс, а как **активного политического агента** с собственным [[Кейс База знаний Геофилософия/3 Понятия/Суверенитет\|Суверенитет]].

## Истоки концепции
- **[[Кейс База знаний Геофилософия/4 Авторы/Латур, Бруно\|Латур, Бруно]]**: Вводит понятие [[Кейс База знаний Геофилософия/3 Понятия/Геоистория\|Геоистория]], где [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земля]] становится действующим лицом в политике.
- **[[Кейс База знаний Геофилософия/4 Авторы/Шмитт, Карл\|Шмитт, Карл]]**: Его работа "[[Кейс База знаний Геофилософия/3 Понятия/Номос земли\|Номос земли]]" является ключевым источником.

## Земля как суверен
- **[[Кейс База знаний Геофилософия/3 Понятия/Суверенитет\|Суверенитет]]**: Землю можно помыслить как суверена в смысле [[Кейс База знаний Геофилософия/4 Авторы/Шмитт, Карл\|Шмитта]]: **"суверен тот, кто принимает решение о [[Кейс База знаний Геофилософия/3 Понятия/Чрезвычайное положение\|Чрезвычайное положение]]"**.
- **Земля и [[Кейс База знаний Геофилософия/3 Понятия/Чрезвычайное положение\|Чрезвычайное положение]]**: [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Земля]] сама **реализует решение о чрезвычайном положении** через глобальные катастрофы, пандемии, вымирания.
- **[[Кейс База знаний Геофилософия/3 Понятия/Анти-законы\|Анти-законы]] Земли**: С [[Кейс База знаний Геофилософия/3 Понятия/Земля\|Землей]] **нельзя заключить договор**, потому что у нее нет предсказуемых "законов". У нее есть "анти-законы" — непредсказуемые, но необходимые события.
